REGISTERS


Name Description Usage
$pc Program Counter - contains the memory address of the next instruction to execute never assign to program counter
$zero always contains the value zero never assign to the $zero register

ARITHMETIC


Instruction Op Code Example Explanation Syntax
Add add add $t0, $t1, $t2 adds $t1 and $t2 and assigns the result to $t0 add destinationRegister, sourceRegister, sourceRegister
Subtract sub sub $t0, $t1, $t2 subtracts $t2 from $t1 and assigns the result to $t0 sub destinationRegister, sourceRegister, sourceRegister
Multiply mult mult $t0, $t1, $t2 multiplies $t1 and $t2 and assigns the result to $t0 mult destinationRegister, sourceRegister, sourceRegister
Divide div div $t0, $t1, $t2 divides $t1 by $t2 and assigns the result to $t0
this computer uses integer division only. e.g., 11 / 2 = 5 		mult destinationRegister, sourceRegister, sourceRegister
Modulus mod mod $t0, $t1, $t2 divides $t1 by $t2 and assigns the REMAINDER to $t0
e.g., if $t1 = 11 and $t2 = 5, then $t0 = 1 		mod destinationRegister, sourceRegister, sourceRegister

LOGIC


Instruction Op Code Example Explanation Syntax
Jump j j __myLabel sets the program counter to the address of __myLabel. j label
Branch if Equal beq beq $t0, $t1 __myLabel if $t0 and $t1 are equal, beq sets the program counter to the address of __myLabel.
if $t0 and $t1 are NOT equal, beq sets the program counter to the next instruction.
beq registerOne, registerTwo, label
Branch if Greater than Zero bgtz bgtz $t0, __myLabel if $t0 is greater than zero, bgtz sets the program counter to the address of __myLabel.
if $t0 is less than or equal to zero, bgtz sets the program counter to the next instruction.
bgtz register, label

LOAD / STORE / MOVE


Instruction Op Code Example Explanation Syntax
Load Word lw lw $t0, 3($t1) takes the data stored at memory address (3 + $t1) and copies it to $t0 lw destinationRegister, offset(sourceAddressRegister)

offset is optional, default value is 0
() characters are required even if there is no offset
Store Word sw sw $t0, 3($t1) takes the data stored in $t0 and copies it to memory address (3 + $t1) sw sourceRegister, offset($destinationAddressRegister)

offset is optional, default value is 0
() characters are required even if there is no offset
Load Address la la $t0, __myLabel sets $t0 to the address of __myLabel la register, label
Load Immediate li li $t0, 999 sets $t0 to a literal value, in this case 999 li register, value
Move move move $t0, $t1 copies $t1 to $t0 move destinationRegister, sourceRegister
